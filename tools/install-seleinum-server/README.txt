
# Install and configure selenium server #

  run the command:

================================================================================

$  sh ./install-selenium-server-2.53.1.sh

================================================================================



Selenium standalone server will downloaded and installed on your localhost or server.

Then selenium will start working after a reboots.

This selenium server will start at the 4445 port number.

You could use the following commands to control the server

================================================================================

sudo /etc/init.d/selenium start

sudo /etc/init.d/selenium stop

sudo /etc/init.d/selenium restart

================================================================================


To Uninstall :

================================================================================

$  sh ./uninstall-selenium-server.sh

================================================================================
Then Reboot.
